# RCV Android and iOS uikits Doc

The RingCentral Client Video SDK makes it easy to do the collaborative audio calling, video calling, and screen sharing based on the RingCentral platform service.

The RingCentral Client Video SDK Android and iOS Components Library supplies client-side state management and reusable UI components for common mobile native interfaces used in audio and video conferencing applications, including: 

- ActionBar
- Actions
  - AudioAction
  - VideoAction
  - MeetingInfoAction
  - ParticipantAction
  - ChatAction
  - RecordAction
  - LeaveAction
